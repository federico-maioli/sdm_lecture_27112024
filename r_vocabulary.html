R vocabulary

Basic R Vocabulary for Data Manipulation

Reading Data

Function Description Example
read.csv() Reads a CSV file into a data frame data <- read.csv("file.csv")
read.table() Reads a table (general text file) data <- read.table("file.txt", header = TRUE)
read_excel() Reads an Excel file (requires readxl pkg) data <- read_excel("file.xlsx")
write.csv() Writes a data frame to a CSV file write.csv(data, "output.csv")

Subsetting Data

Function/Operator Description Example
$ Selects a single column in a data frame data$column_name
[ ] General indexing for rows and columns data[1, ] (first row), data[, 2] (second col)
subset() Subsets data based on a condition subset(data, column_name > 5)
filter() Subsets rows based on conditions (dplyr) filter(data, column_name > 5)
select() Selects columns by name (dplyr) select(data, col1, col2)

Basic Data Manipulation

Function Description Example
mutate() Creates/modifies columns in a data frame (dplyr) mutate(data, new_col = col1 * 2)
arrange() Sorts data by columns (dplyr) arrange(data, col1)
rename() Renames columns in a data frame (dplyr) rename(data, new_name = old_name)
summarise() Summarizes data (dplyr) summarise(data, mean_col = mean(col1))
group_by() Groups data by one or more columns (dplyr) group_by(data, col1)

Basic Plotting (Using ggplot2)

Function Description Example
ggplot() Initializes a plot ggplot(data, aes(x = col1, y = col2))
geom_point() Adds a scatter plot layer + geom_point()
geom_line() Adds a line plot layer + geom_line()
geom_bar() Adds a bar plot layer + geom_bar(stat = "identity")
facet_wrap() Creates a grid of plots based on a factor + facet_wrap(~ factor_col)

Other Useful Functions

Function Description Example
str() Shows the structure of an object str(data)
summary() Provides a summary of a data frame summary(data)
head() / tail() Shows the first/last few rows of a data frame head(data)
dim() Returns the dimensions of a data frame dim(data)
names() Returns the column names of a data frame names(data)
